# Titanic Dataset - Exploratory Data Analysis (EDA)

## Objective
Perform exploratory data analysis on the Titanic dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Tasks Performed
1. Generated summary statistics
2. Created histograms
3. Created boxplots
4. Generated correlation heatmap
5. Created pairplot
6. Identified patterns and trends

## Key Findings
- Age distribution concentrated around young adults.
- Fare contains outliers.
- Passenger class influences fare.
- Some features show correlation with survival.
